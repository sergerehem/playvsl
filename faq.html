1) Como embedar o PlayVSL?

Copie este bloco e cole na sua página. Troque apenas a URL do YouTube e o link do botão.

<div id="playvsl"></div>
<script src="https://playvsl.digi6.com.br/playvsl.js"></script>
<script>
  PlayVSL.init({
    container:'#playvsl',
    youtubeUrl:'https://youtu.be/wqGiHRWeTR0',
    buttonUrl:'https://playvsl.digi6.com.br/',
    buttonText:'QUERO MEU SNIPPET AGORA - É GRÁTIS',
    buttonShowAtSeconds:10
  });
</script>

2) Quais eventos existem?

Eventos servem para medir comportamento do usuário (play, pausa, clique no CTA, conclusão etc.). Você pode usar callbacks específicos ou centralizar tudo em onEvent.

  • onReady
  • onPlay
  • onFirstPlay
  • onResume
  • onRestart
  • onPause
  • onProgress (intervalos e marcos 25/50/75/100)
  • onCTAView
  • onCTAClick
  • onComplete
  • onError
PlayVSL.init({
  ...,
  onEvent: (payload) => {
    // Envie para n8n, GA4, API etc
    console.log(payload.event, payload);
  }
});

3) Como fazer “sumir elementos acima do vídeo no play”?

Use este padrão quando quiser focar o usuário no vídeo. No play, você esconde os blocos de cima; no pause, volta a mostrar.

PlayVSL.init({
  ...,
  onPlay: () => document.body.classList.add('video-playing'),
  onPause: () => document.body.classList.remove('video-playing')
});
.video-playing .headline,
.video-playing .preheader {
  display:none;
}

4) Como enviar eventos para n8n?

Esta é a forma mais simples de começar a ter estatísticas sem criar backend complexo: o PlayVSL envia eventos para um webhook e o n8n grava onde você quiser.

const WEBHOOK = 'https://SEU-N8N/webhook/playvsl-events';

PlayVSL.init({
  ...,
  onEvent: (payload) => {
    fetch(WEBHOOK, {
      method:'POST',
      headers:{'Content-Type':'application/json'},
      body: JSON.stringify(payload),
      keepalive: true
    }).catch(()=>{});
  }
});

5) Dúvidas comuns

Respostas rápidas para os problemas mais frequentes no dia a dia.

A barra chegou em 100% antes do fim

Já corrigido: ela converge para 100% no final real do vídeo.

Quero CTA imediato

Use buttonShowAtSeconds: 0.

PlayVSL já oculta o iframe no pause/final para evitar esses overlays e manter foco na conversão.
